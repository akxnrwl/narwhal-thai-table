Roadmap

เราอยู่ตรงไหน และเหลืออะไรอีก
Phase 1 · ตอนนี้80%

Pilot on Narwhalพิสูจน์ผลด้วยร้านจริง

  • ✓Aileen chatbot ขึ้นไลฟ์ (Sonnet 4.6)
  • ✓Analytics /stats เก็บข้อมูลจริง
  • ✓อีเมลแจ้งเตือนแยกช่องทาง
  • ●เก็บสถิติให้ครบเพื่อทำเคสขาย
Phase 2 · ถัดไป10%

Productizeทำให้ขายหลายร้านได้

  • ●Multi-tenant core แยกข้อมูลรายร้าน
  • ○หน้า onboarding เปิดร้านใหม่
  • ○ล็อกอิน + แดชบอร์ดของเจ้าของร้าน
  • ○ระบบ เก็บเงิน Stripe
Phase 3 · อนาคต5%

Expandเพิ่มบริการเต็มแพลตฟอร์ม

  • ○Voice agent รับสายโทรศัพท์
  • ○Lead engine + CRM
  • ○Toast / POS integration
  • ○Back-office หลังบ้านครบวงจร
